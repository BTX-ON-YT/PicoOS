# PicoOS
An OS for boards with the RP2040 Processor, Currently Under Development.

# Compatibility
Boards with the RP2040 processor and Micropython.

# Setup
Type in the "setup" command in PicoOS.

# Help
Use the "help" command in PicoOS.
